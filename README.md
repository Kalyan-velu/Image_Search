# Image_Search
>A image search website that uses pixabay api to fetch images

## To use
* Install Dependencies using
>using npm
```npm
  cd client && npm i && cd ../server npm i
```
>using yarn
```yarn
  cd client && yarn install && cd ../server yarn install
```
* To Start the client && server side app
>using npm
```npm
  npm start
```
>using yarn
```yarn
  yarn start
```
